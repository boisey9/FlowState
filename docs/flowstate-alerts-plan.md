# FlowState Alerts Plan

This document tracks the planned safe alert layer for FlowState.

## Phase 1

Add in-app alert status for WATCH LONG, WATCH SHORT, READY LONG, READY SHORT, NO ALERT, and DATA WARNING.

## Phase 2

Add optional browser notifications while the app is open.

## Phase 3

Add scheduled Supabase/Telegram alerts later without changing the current decision engine.
